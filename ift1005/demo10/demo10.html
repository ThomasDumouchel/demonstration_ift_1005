Node.js

Durant la dernière démonstration, nous avons vue comment écrire un serveur avec Javascript. Pour ce faire, nous avons utilisé le module http de Node.js. Node.js est un environnement d'exécution Javascript qui permet d'exécuter du code Javascript en dehors d'un navigateur web. Nous avons aussi vu comment utiliser npm pour installer des nouveaux packages dans notre projet. Nous avons utilisé le package express pour créer un serveur web plus facilement. Nous avons aussi vu comment utiliser le package nodemon pour redémarrer automatiquement notre serveur.

AJAX, JSON et XML

AJAX est un acronyme pour Asynchronous Javascript And XML. AJAX permet d'envoyer des requêtes HTTP à un serveur web et de recevoir des données en réponse. Les données peuvent être au format XML, JSON ou autre. Nous avons vu comment utiliser AJAX pour envoyer des requêtes HTTP à un serveur web et recevoir des données en réponse. Nous avons aussi vu qu'on peut utiliser AJAX pour envoyer des données au serveur web.

JSON est un format de données qui permet de représenter des objets Javascript. JSON est un format de données textuel. JSON est souvent utilisé pour échanger des données entre un serveur web et un client web. Nous avons vu comment utiliser JSON pour échanger des données entre un serveur web et un client web.

XML est un autre format de données qui permet de représenter des objets. XML est un format de données textuel. XML est souvent utilisé pour échanger des données entre un serveur web et un client web (néanmoins, JSON est plus populaire).
