Expression régulière: Regex

Un bon tutorial
Un lien pour pratiquer et tester

C'est quoi?

Une méthode pour chercher à travers un texte un string suivant un certain pattern.
L'expression régulière (regex), c'est le pattern.

C'est comment?

Voici les règles pour écrire un regex:

  1. Format

    Voici le format: / ...regex... /g
    On écrit notre pattern entre les deux /
    Les charactères qui viennent après le deuxième / sont des flags. Ici, g

  2. Quantificateurs

    Toujours appliqué sur ce qui est à sa gauche

    • + : 1 ou plus
    • ? : 0 ou 1
    • * : 0 ou plus
    • {min, max} : entre min inclusivement et max inclusivment
  3. Ensemble et range
    • [ab] : match le char a ou b
    • [ab 2] : match le char a ou b ou " " (l'espace) ou 2
    • [a-g] : match les char de a à g. Équivalent à [abcdefg]
      [A-G] : match les char de A à G. Équivalent à [ABCDEFG]
      [0-9] : match les char de 0 à 9. Équivalent à [0123456789]
  4. Groupement
    • (koala|panda) : match soit koala soit panda
    • (abc){2,3} : match abcabc ou abcabcabc
  5. Début de ...
    • texte. Avec ^
    • mot. Avec \b au début
      ex. pattern: \bis[a-zA-Z]* string = "This island is beautiful" matched_strings = ["island", "is"], mais pas "This"
  6. Fin de ...
    • texte. Avec $
    • mot. Avec \b à la fin
      ex. pattern: [a-zA-Z]*is\b string = "This island is beautiful" matched_strings = ["This", "is"], mais pas "island"
  7. Chars spéciaux

    Pour les ajouter dans nos pattern, mettre \ en préfixe.
    ex. \+ ou \.

    Liste: ^ { } ( ) [ ] + * . $ | ^ \

  8. Metas séquences
    • . : tous, sauf le saut de ligne
    • \s : tous les espaces
    • \d : tous les chiffres
    • \w : tous les chars alpanumériques et underscore (_)
