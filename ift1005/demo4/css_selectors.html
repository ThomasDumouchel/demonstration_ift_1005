Sélecteur CSS: comment appliquer un style sur nos éléments html

Liste exhaustive des sélécteurs 
            
                /* 
                Pour accèder une classe, c'est un . suivit du nom de la classe

                Appliquera le style background-color: blue; à tous les éléments de notre code 
                qui possède l'attribut class="className1"
                */
                .className1 {
                    background-color: blue;
                }

                /* 
                Pour accèder un élément avec un certain id, c'est un # suivit du nom du id

                Appliquera le style background-color: blue; à tous les éléments de notre code 
                qui possèdent l'attribut class="className1"
                */
                #idname {
                    background-color: blue;
                }

                /* 
                Appliquera le style background-color: blue; à tous les éléments de notre code 
                qui possèdent l'attribut class avec className1 et className2
                */
                .className1.className2 {
                    background-color: blue;
                }

                /* 
                Appliquera le style background-color: blue; à tous les éléments de notre code 
                qui sont décendant d'un élément de classe className1 et qui possède la classe className2
                */
                .className1 .className2 {
                    background-color: blue;
                }

                /* 
                Appliquera le style background-color: blue; à absolument tous les éléments
                */
                * {
                    background-color: blue;
                }

                /* 
                Appliquera le style background-color: blue; à tous les div

                Peut-être utilisé avec tous les éléments
                */
                div {
                    background-color: blue;
                }

                /* 
                Appliquera le style background-color: blue; à tous les div ayant la classe className1
                */
                div.className1 {
                    background-color: blue;
                }

                /* 
                Appliquera le style background-color: blue; à tous les div et tous les p
                */
                div, p {
                    background-color: blue;
                }

                /* 
                Appliquera le style background-color: blue; à tous les p descendant d'un div
                */
                div, p {
                    background-color: blue;
                }

                /* 
                Appliquera le style background-color: blue; à tous les p qui sont descendant direct d'un div
                */
                div > p {
                    background-color: blue;
                }

Exemple:

Soit le code html suivant

            
            <div id="root">

                <div class="children-of-root" id="first-child">
                    Texte dans div 1
                    <a href="">Lien 1 de div 1</a>
                    <div>
                        <a href="">Lien 2 de div 1</a>
                    </div>
                </div>

                <p class="children-of-root">
                    Premier paragraphe
                </p>

                <div class="children-of-root" id="second-child">
                    Texte dans div 2
                    <a href="">Lien 1 de div 2</a>
                    <div>
                        <a href="">Lien 2 de div 2</a>
                    </div>
                </div>

                <p class="children-of-root">
                    Deuxieme paragraphe
                </p>

            </div>
