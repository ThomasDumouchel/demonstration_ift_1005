L'arbre d'élément html d'une page web

C'est la manière que notre code est stocké dans un browser. Il est utile de comprendre ça pour savoir comment accèder à un élément html puis de le modifier, soit avec CSS ou avec Javascript.

Soit l'exemple de code html suivant:

            
                <html>
                    <head>
                        <link rel="stylesheet" href="">
                        <title>Titre...</title>
                    </head>
                    <body>
                        <header>
                            ...
                        </header>
                        <main>
                            <section>
                                <h1>
                                    Titre de ma section...
                                </h1>
                                <div>
                                    Du contenu...
                                </div>
                            </section>
                        </main>
                        <footer>
                            ...
                        </footer>
                    </body>
                </html>
