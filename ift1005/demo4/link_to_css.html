Ajouter une feuille de style à notre page HTML

Tuto sur linker un fichier css à notre fichier index.html. Commencer à 4:20.

C'est facile! On doit simplement créer un fichier CSS et le linker grâce à une balise link

La partie qui peut être compliqué, c'est de linker. Voici un exemple expliquant les différentes manières de linker: 

            
                Soit notre répertoire monProject ayant la forme suivante:

                monProject 
                    CSS
                        mesStyles.css
                    ressources
                        monImage.png
                    autreRepertoire
                        page2.html
                        encoreUnAutreRepertoire
                            page3.html
                    index.html
                    autreFeuilleDeStyle.css

                ex1. Je veux attacher la feuille de style 'mesStyles.css' à mon index.html

                    Pour ce faire, dans le <head> de mon fichier index.html, j'ajoute 
                    la ligne de code suivante:

                    <link rel="stylesheet" href="CSS/mesStyles.css"/>

                    Si je veux attacher mon autreFeuilleDeStyle.css, j'ajoute:

                    <link rel="stylesheet" href="autreFeuilleDeStyle.css"/>

                    (Ne rien mettre indique que le lien commence au niveau de notre fichier index.html)

                ex2. Je veux attacher la feuille de style 'mesStyles.css' à mon page2.html

                    Pour ce faire, dans le <head> de mon fichier page2.html, j'ajoute 
                    la ligne de code suivante:

                    <link rel="stylesheet" href="../CSS/mesStyles.css"/>

                    Si je veux attacher mon autreFeuilleDeStyle.css, j'ajoute:

                    <link rel="stylesheet" href="../autreFeuilleDeStyle.css"/>

                    (Mettre "../" nous ramène un répertoire en arrière)

                ex3. Je veux attacher la feuille de style 'mesStyles.css' à mon page3.html

                    Pour ce faire, dans le <head> de mon fichier page3.html, j'ajoute 
                    la ligne de code suivante:

                    <link rel="stylesheet" href="/CSS/mesStyles.css"/>

                    (Mettre seulement "/" nous ramène à la racine de notre projet)
