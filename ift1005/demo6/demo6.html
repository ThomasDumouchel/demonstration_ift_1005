Résumé de la dernière démonstration du lundi 31 octobre

Durant la dernière démonstration, j'ai, avec les élèves présents (mes favoris dorénavant) démontré comment utiliser CSS pour faire la page produit de notre TP2. Nous avons vu:

  • Ré-organizer les éléments de style display: block (i.e. ceux qui ajoute un saut de ligne) pour qu'ils s'affiche sur la même rangée. Pour faire ceci, j'utilise la propriété CSS display: flex. Apprennez à l'utiliser, parce que c'est la commande la plus utile en CSS (selon moi, et je suis un expert).
    • Pour mettre le sidebar et le contenu sur la même rangée
    • Pour mettre les éléments de la navigation sur la même rangée
    • Pour centrer des éléments à l'intérieur d'un autre élément
  • Appliquer du style quand notre souris est sur un élément. Pour faire ceci, j'utilise la pseudo-classe CSS :hover.
  • Alligner les éléments de manière custom. Pour faire ceci, j'utilise la propriété CSS position ainsi que les propriétés top, left, right, bottom.
    • Pour créer l'îcone shopping cart avec le petit nombre dans un cercle rouge
    Allez voir la documentation sur W3School pour plus d'information.
  • Durant la démonstration, mes étudiants favoris ont aussi vu comment utiliser le developer-tool de Chrome (qui est pareil pour tout les navigateurs) pour trouver les éléments HTML et les styles qui leur sont appliqués.
