Comment ajouter un script javascript à notre site web

On ajoute un script javascript à notre site web en utilisant la balise <script>.
Exemple: <script src="index.js"></script>

Comme le code HTML est lu ligne par ligne de haut en bas, il faut placer notre tag <script> à la fin du body.
Si on l'ajoute avant, le script sera exécuter avant que le reste du code HTML soit lu. Donc tous accès à des éléments HTML ne fonctionnera pas, ...
