jQuery

jQuery est une bibliothèque JavaScript qui permet de simplifier l'écriture de code JavaScript.

C'est-à-dire que jQuery nous permet d'accéder aux éléments HTML de la page web et de les manipuler plus facilement, ou du moins de façon plus concise, que si on utilisait du code JavaScript pur (ce qu'on a fait la semaine passé).

On va l'utiliser pour ajouter des éléments HTML à notre page web, pour les modifier, pour les supprimer. On va aussi l'utilisier pour réagir à des événements (par exemple, quand on clique sur un bouton).

Set-up jQuery

Pour utiliser jQuery, il faut d'abord inclure la bibliothèque jQuery dans notre page web. On peut le faire en téléchargeant la bibliothèque et en l'incluant dans notre projet, ou en utilisant un CDN (Content Delivery Network). Un CDN est un réseau de serveurs qui contiennent des fichiers, et qui permettent de les distribuer rapidement à travers le monde. On va utiliser un CDN pour inclure jQuery dans notre page web.
Écrire le code suivant dans notre page web: <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.1/jquery.min.js"></script>

Sélectionner un élément de notre page

Pour sélectionner un élément de notre page, on utilise la fonction $(selector).
Le selector est une chaîne de caractères qui permet de sélectionner un élément de notre page. Par exemple:

  • Si on veut sélectionner l'élément HTML qui a l'identifiant produits, on écrira $("#produits"). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementById("produits").
  • Si on veut sélectionner tous les éléments HTML qui ont la classe produit, on écrira $(".produit"). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementsByClassName("produit"). Note: lorsque on sélectionne plusieurs éléments, on recevra un tableau d'éléments. Le traitement sera donc un peu différent.
  • Si on veut sélectionner tous les éléments HTML qui ont le tag h1, on écrira $("h1"). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementsByTagName("h1"). Note: lorsque on sélectionne plusieurs éléments, on recevra un tableau d'éléments. Le traitement sera donc un peu différent.
  • Si on veut sélectionner tous les tag a dans un div qui a l'identifiant contact, on écrira $("#contact a").

Appliquer du style sur nos éléments

Pour appliquer du style sur nos éléments, on utilise la fonction css().

  • Si on veut appliquer un style background-color à l'élément HTML qui a l'identifiant produits, on écrira $("#produits").css("background-color", "red"). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementById("produits").style.backgroundColor = "red".
  • Pour appliquer la couleur rouge à tous les éléments HTML qui ont la classe produit, on écrira $(".produit").css("color", "red"). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme 
                                let elements = document.getElementsByClassName("produit"); 
    
                            for (let i = 0; i < elements.length; i++) { elements[i].style.color = "red"; }
  • Si on veut appliquer plusieurs style à un élément HTML, on peut écrire quelque chose comme 
                                $("#produits").css({ 
                                "background-color": "red", 
                                "color": "white" 
                            })

Ajouter une classe

Pour ajouter une classe à un élément HTML, on utilise la fonction addClass().
Par exemple, pour ajouter la classe make-beautiful à l'élément HTML qui a l'identifiant produits, on écrira $("#produits").addClass("make-beautiful")

Supprimer une classe

Pour supprimer une classe d'un élément HTML, on utilise la fonction removeClass().
Par exemple, pour supprimer la classe make-beautiful à l'élément HTML qui a l'identifiant produits, on écrira $("#produits").removeClass("make-beautiful")

Toggle une classe

Pour ajouter une classe à un élément HTML si elle n'est pas présente, et la supprimer si elle est présente, on utilise la fonction toggleClass().
Par exemple, pour ajouter la classe make-beautiful à l'élément HTML qui a l'identifiant produits, on écrira $("#produits").toggleClass("make-beautiful") Pour le faire en javascript, il faut l'écrire au long: 

                    let element = document.getElementById("produits"); 

                    if (element.classList.contains("make-beautiful")) { 

                        element.classList.remove("make-beautiful"); 

                    } else { 

                        element.classList.add("make-beautiful"); 

                    }

Changer le html

Pour changer le html d'un élément, on utilise la fonction html().

  • Si on veut changer le html de l'élément HTML qui a l'identifiant produits, on écrira $("#produits").html("Nouveau html"). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementById("produits").innerHTML = "Nouveau html".
  • Si on veut changer le html de tous les éléments HTML qui ont la classe produit, on écrira $(".produit").html("Nouveau html"). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme 
                                let elements = document.getElementsByClassName("produit"); 
    
                            for (let i = 0; i < elements.length; i++) { elements[i].innerHTML = "Nouveau html"; }

Changer le texte

Pour changer le texte d'un élément, on utilise la fonction text().
Si on veut changer le texte de l'élément HTML qui a l'identifiant produits, on écrira $("#produits").text("Nouveau texte"). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementById("produits").innerText = "Nouveau texte".

Cacher un élément (pas le supprimer, juste l'ôter de la page)

Pour cacher un élément, on utilise la fonction hide().
Par exemple, si on veut cacher l'élément HTML qui a l'identifiant produits, on écrira $("#produits").hide(). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementById("produits").style.display = "none"

Pour afficher un élément, on utilise la fonction show().
Par exemple, si on veut afficher l'élément HTML qui a l'identifiant produits, on écrira $("#produits").show(). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementById("produits").style.display = "block" ou document.getElementById("produits").style.display = "inline" ou ...

Supprimer un élément

Pour supprimer un élément, on utilise la fonction remove().
Par exemple, si on veut supprimer l'élément HTML qui a l'identifiant produits, on écrira $("#produits").remove(). Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme document.getElementById("produits").remove()

Réagir à un événement click

Pour réagir à un événement click, on utilise la fonction click().
Par exemple, si on veut réagir au click sur tous les éléments HTML qui ont la classe produit, on écrira 

                    $(".produit").click(function() { $(this).remove() })
Si on utilisait du JavaScript pur, on devrait écrire quelque chose comme 
                    let elements = document.getElementsByClassName("produit"); 

                    for (let i = 0; i < elements.length; i++) { 
                        elements[i].addEventListener("click", function(event) {) { 
                            event.target.remove(); 
                        });
                    }

Note: attraper un élément d'événement dans notre fonction de réaction

Dans le code ci-dessus, on a utilisé $(this) pour réagir au click sur tous les éléments HTML qui ont la classe produit.
$(this) permet de réagir à l'élément HTML qui a déclenché l'événement.
Si on utilisait du JavaScript pur, doit indiquer que notre fonction doit prendre un paramètre event et que l'élément HTML qui a déclenché l'événement est event.target. 

                    let elements = document.getElementsByClassName("produit"); 

                    for (let i = 0; i < elements.length; i++) { 
                        elements[i].addEventListener("click", function(element) {) { 
                            element.target.remove(); 
                        });
                    }
Ceci n'empêche pas de faire la même chose dans le cas de jQuery: 
                    $(".produit").click(function(event) { event.target.remove() })

Réagir à un événement double click

Pour réagir à un événement double click, on utilise la fonction dblclick().
Par exemple, si on veut réagir au double click sur tous les éléments HTML qui ont la classe produit, on écrira 

                    $(".produit").dblclick(function() { $(this).toggleClass("bg-red") })

Réagir à hover

Pour réagir à un événement hover, on utilise la fonction hover().
Par exemple, si on veut réagir au hover sur tous les éléments HTML qui ont la classe produit, on écrira 

                    $(".produit").hover(function() { $(this).toggleClass("bg-red") })

On a pas toute la journée: Tous les événements accessibles par Javascript pur le son aussi par jQuery

Pour réagir aux événements, on peut aussi utiliser une notation plus proche de Javascript, soit avec on() et le nom de l'événement. Par exemple:

  • $(".produit").on("click", function() { $(this).remove() })
  • $(".produit").on("dblclick", function() { $(this).toggleClass("bg-red") })
  • ...
