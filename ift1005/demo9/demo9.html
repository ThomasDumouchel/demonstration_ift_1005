SVG

SVG est un format de fichier vectoriel. Il est utilisé pour représenter des images vectorielles. Il est possible de créer des images SVG à l'aide d'un éditeur de texte ou d'un éditeur graphique.

Les images SVG peuvent être intégrées dans une page web directement à l'aide de la balise <svg>.
Exemple: 

                <svg width="100" height="100">
                  <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
                </svg>

Les images SVG peuvent être intégrées dans une page web à l'aide de la balise <img>.
Exemple: <img src="mysvg.svg">
